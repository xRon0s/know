Raspberry Pi Server Setup Guide

このサイトは、Raspberry Piを使用してセキュアなWebサーバーを構築するための手順を、備忘録としてまとめたものです。

前提条件

  • Raspberry Pi 4 Model B
  • microSDカード (32GB以上推奨)
  • ディスプレイ、キーボード、マウス
  • インターネット接続環境 (有線または無線)
  • 作業用PC (SSH接続用)

ページ一覧

  • Goals

    このサーバー構築の最終的な目標を定義します。

  • Image

    サーバーの全体構成図と通信の流れを視覚化します。

  • Preparation

    OSのインストールから初期設定までを解説します。

  • Authority

    ユーザー権限、SSH鍵認証、ファイアウォール設定について解説します。

  • Web

    Fail2banによるセキュリティ強化とNginxの導入について解説します。

  • Troubleshooting

    構築中によくある問題とその解決策をまとめます。
