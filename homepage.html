Welcome

Raspberry Pi Server Setup Guide

このサイトは、Raspberry Piを使用してセキュアなWebサーバーを構築するための手順を、備忘録としてまとめたものです。

前提条件

  • Raspberry Pi 4 Model B
  • microSDカード (32GB以上推奨)
  • ディスプレイ、キーボード、マウス
  • インターネット接続環境 (有線または無線)
  • 作業用PC (SSH接続用)

ページ一覧

  • 目標

    このプロジェクトで達成する具体的な目標について説明します。

  • 全体構成図

    サーバーの構成とネットワークの流れを視覚的に解説します。

  • OS準備

    Raspberry Pi OSのインストールと初期設定の手順です。

  • 権限/SSH/FW設定

    ユーザー権限、SSHの鍵認証、ファイアウォールの設定方法です。

  • 堅牢化/Webサーバー

    Fail2banによる不正アクセス対策とNginxの導入手順です。

  • トラブルシューティング

    構築中に発生しがちな問題とその解決策をまとめています。
